## Project Overview
---
A comprehensive PostgreSQL database tracking the complete 2024-25 NBA season statistics.

## Functional Requirements
---
1. Store all 30 NBA teams with conference/division affiliations
2. Track 450+ players with biographical and statistical data
3. Record per-game averages for all major statistics
4. Support complex analytical queries
5. Demonstrate database normalization principles

## Technical Requirements
---
1. **Database**: PostgreSQL 12+
2. **Normalization**: 3rd Normal Form compliance
3. **Relationships**: Proper foreign key constraints
4. **Performance**: Appropriate indexing strategy
5. **Backup**: Documented backup/restore procedures

## Deliverables
---
- ER Diagram and normalized schema
- PostgreSQL implementation with sample data
- SQL queries (basic and advanced)
- Transaction examples
- Indexing strategy
- Backup/restore documentation
- GitHub repository with complete project

## Sample Data
---
- Minimum: 4 teams, 20 players with stats
- Recommended: All 30 teams, 100+ players
- Realistic 2024-25 season statistics
